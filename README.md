# Understanding Analysis 习题解答
Understanding Analysis 是 Stephen Abbott 编写的数学分析教材。笔者在自学过程中试着进行习题解答。


## 解答内容与查看方式

- 内容以第一版习题为主，部分习题挑选自第二版。
- 在 [我的个人网站](https://calculus1437.github.io/) 上可以在线阅读解答与其他内容，不定期更新。
- 同时会在 Release 中发布 PDF 文件。

## 提示

- 习题块格式使用 VSCode 插件 [Markdown Preview Enhanced](https://marketplace.visualstudio.com/items?itemName=shd101wyy.markdown-preview-enhanced)，在 VSCode 上安装此插件以获得 Markdown 文件良好的阅读体验。（设置里搜索 math rendering option，将渲染引擎换成 MathJax 可以获得最佳体验）

- markdownstyle.less 是 Markdown Preview Enhanced 渲染风格的配置文件，将它重命名并替换 C:\Users\用户名\.crossnote\style.less 下的同名文件即可实现同样的渲染效果。
